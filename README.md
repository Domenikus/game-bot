# Game bot

## Setup:

```
composer install
php8.0 game-bot migrate
```

## Run:

### Bot:

```
php8.0 game-bot run
```

### Menu:

```
php8.0 game-bot menu
```

## Build for production

```
php8.0 game-bot app:build
```

## Bot chat commands

### Register

#### Apex

```
!register|apex|{name}|{platform}
```

### Unregister

```
!unregister|{?game}
```

### Update

```
!update
```
